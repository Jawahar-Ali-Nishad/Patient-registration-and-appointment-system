Project Title:
Patient Registration and Appointment System.
Submitted By:
Jawahar Ali Nishad S
Date:

Objectives:
The main goal of this project is to create a simple web page that allows users to register new patients and book doctor appointments easily.
Tools Used:
"	HTML, CSS, and JavaScript.
"	Bootstrap (for design).
"	Local Storage (for data storage).
"	VS Code (for coding).
Functions:
"	Add new patient details.
"	Book appointments with department and doctor.
"	 Display all booked appointments.
"	Edit or delete any records.
"	Store data even after page refresh using local storage.


Page Design:
The page has a navigation bar at the top with logo and two links to display patient records and all appointments. When you look below, there is a form to enter patient details and appointment details. On each form there is a button to save the entered details.
References:
For design, I visited drible.com to get an idea for web design, Bootstrap for logos, Concept understand by viewing videos on YouTube and debugging with DeepSeek and ChatGPT.
